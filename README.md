# 五子棋游戏 (Gobang Game)

一个基于纯前端技术实现的五子棋小游戏。

## 项目简介

五子棋是一种两人对弈的棋类游戏，双方分别使用黑白两色的棋子，轮流下在棋盘上，先在横、竖、斜任意方向连成五子者获胜。

## 功能特性

- 15x15 标准棋盘
- 黑白双方轮流下棋
- 自动判断胜负
- 悔棋功能
- 重新开始游戏
- 实时显示当前执棋方
- 获胜提示动画

## 技术栈

- HTML5
- CSS3
- JavaScript (原生)

## 如何运行

### 方式一：直接打开

双击 `index.html` 文件即可在浏览器中运行游戏。

### 方式二：使用本地服务器

```bash
# 使用 Python
python -m http.server 3000

# 使用 Node.js
npx serve .
```

然后访问 http://localhost:3000

## 游戏规则

1. 黑棋先行，双方轮流落子
2. 棋子落在棋盘交叉点上
3. 先在横、竖、斜任意方向连成五子者获胜
4. 棋盘下满则平局

## 操作说明

- 点击棋盘交叉点落子
- 点击"重新开始"按钮重置游戏
- 点击"悔棋"按钮撤销上一步
